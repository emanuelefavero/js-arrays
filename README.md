# JS Arrays

Esercizio sugli array in JavaScript dal mio corso di Web Development.

## Obiettivo

Partendo dal codice fornito, completare tutte le task richieste negli script presenti.

Istruzioni principali:

- svolgere gli esercizi dentro i file `script.js`;
- usare DevTools e `console.log` per verificare il risultato;
- completare anche gli script bonus, se si finisce in anticipo l'esercizio base;
- non usare in questa fase metodi come `map`, `filter`, `includes` e simili;
- risolvere le task aiutandosi con il ciclo `for`, dove richiesto.

## Soluzioni

Controlla i file degli script per la soluzione:

- `01-arrays/script.js`
- `02-arrays-bonus/script.js`

## Decisioni Tecniche

- Nel primo esercizio ho usato i metodi base degli array già introdotti nella lezione. Nel bonus ho preferito usare cicli `for` per seguire le indicazioni dell'esercizio e esercitarmi sul costrutto.
- Ho aggiunto `console.log` con label per migliorare la leggibilità dei risultati nei DevTools.
